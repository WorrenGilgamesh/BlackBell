plazo   sustantivo masculino 

1

Espacio limitado de tiempo durante el cual se tiene que hacer algo.

Le dieron a Osvaldo un plazo de dos días para entregar el proyecto.  

 //  locución adverbial a corto plazo.

Dentro de un período de tiempo breve.

Las ganancias de este negocio se obtendrán a corto plazo.  

 //  locución adverbial a largo plazo.

Dentro de un período de tiempo extenso.

Se pretende construir esa obra a largo plazo.  

2

Cada una de las partes en que se acuerda dividir el pago total de algo.

Mayra ya pagó el útimo plazo de sus muebles.  

 //  locución adverbial a plazos.

En varios pagos.

Rosa compró un televisor a plazos.  

pla-zo; plazos (plural)
