elaborar   verbo transitivo  

1

Preparar una cosa con materias diferentes para crear algún producto.

Rosa elaboró una pasta de bocaditos para la fiesta.  

sinónimos:  producir,  procesar,  crear,  hacer  

2

Transformar el organismo las sustancias para producir otras necesarias para su funcionamiento.

El páncreas elabora jugos que ayudan a la digestión.  

3

Crear o producir una cosa nueva.

 Elaboramos un plan de actividades para las vacaciones.  

sinónimos:  gestar  

e-la-bo-rar; elaborado (participio pasivo)
